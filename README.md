# Sales Dashboard – Power BI Project

## 📌 Overview
The Sales Dashboard is an interactive Power BI project designed to analyze sales transactions, payment methods, product performance, sales types, profit percentage, and category-wise selling trends.

This dashboard provides business insights through KPI cards, charts, slicers, and visual analytics for better decision-making.

---

## 📊 Dashboard Features

### KPI Cards
- Total Sales
- Total Profit
- Total Profit %
- Total Quantity

### Interactive Filters
- Quarter Filter
- Year Filter
- Month Filter
- Payment Type Filter
- Sales Type Filter

### Visual Reports
- Total Selling by Sales Type
- Total Selling Price by Payment Mode
- Total Selling Price by Product
- Total Selling Price by Day
- Total Selling Price by Category

---

## 🛠️ Tools & Technologies Used
- Power BI
- Power Query
- DAX
- Excel / CSV Dataset

---

## 📈 Insights Generated
- Direct Sales contributes the highest revenue.
- Online payment mode has major transaction share.
- Product41 records the highest selling price.
- Sales fluctuate across different days of the month.
- Category-wise analysis helps identify profitable segments.

---

## 🚀 Project Objectives
- Analyze overall sales performance.
- Track product and category-wise revenue.
- Understand customer payment behavior.
- Visualize daily sales trends.
- Improve business decision-making using BI tools.
